# AI Agent 基础概念

## 一句话理解
AI Agent = 能**自主决策和行动**的 AI，不只是"说话"，而是"干活"。

## 核心组件
```
┌────────────────────────────────────────┐
│              AI Agent                  │
├────────────────────────────────────────┤
│  1. LLM（大脑）    – 决策和推理         │
│  2. Tools（工具）  – 执行（API/DB/搜索）│
│  3. Memory（记忆） – 短期+长期记忆      │
│  4. Planning（规划）– 任务分解和反思    │
└────────────────────────────────────────┘
```

## 和普通 LLM 的区别
| LLM | Agent |
|-----|-------|
| 只能生成文字 | 能调用工具、执行动作 |
| 无状态 | 有记忆，能记住上下文 |
| 单次回复 | 多步推理，直到任务完成 |

## 面试话术
> "Agent 和普通 LLM 的区别在于：LLM 只能说话，Agent 能干活。Agent 通过调用工具（搜索、API、数据库）完成实际任务。"

## 实际例子
我（沈幼楚）就是一个 Agent：
- LLM → DeepSeek 负责理解和生成
- Tools → 搜索、看图、读文件
- Memory → SQLite 记忆库
- Planning → ReAct 循环，思考→行动→观察
